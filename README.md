# Unified Operations Command Center

MeauxOS Dashboard - A unified operations command center with glassmorphic design.

## Features

- **Glassmorphic Sidebar**: Clean frosted glass design with light blue/white aesthetics
- **Dark Space Theme**: Professional dark theme for main content area
- **Modular Architecture**: Multiple integrated modules (MeauxWork, MeauxSQL, MeauxTalk, MeauxMail)
- **Kanban Board**: Task management with drag-and-drop functionality
- **Responsive Design**: Mobile-friendly with hamburger menu
- **Starfield Background**: Animated starfield for visual depth

## Modules

### Core Modules
- **Dashboard**: Overview and system status
- **MeauxWork**: Project management and kanban boards
- **Analytics**: Real-time analytics and reporting
- **Team**: Team management and collaboration
- **Agent_Sam**: AI assistant interface with chat, task automation, and settings

### Content & Media
- **Documents**: Document management
- **Photos**: Photo library
- **Gallery**: Media gallery
- **Media**: Video and media management

### Infrastructure
- **Cloud**: Cloud storage integration (R2)
- **CAD Studio**: 3D modeling workstation
- **DevOps**: Development operations

## Tech Stack

- Pure HTML/CSS/JavaScript
- Inter & JetBrains Mono fonts
- CSS Variables for theming
- Glassmorphism design
- Responsive grid layout

## Deployment

Open `dashboard.html` in a modern web browser to view the dashboard.

For production deployment:
1. Upload to R2 storage or static hosting
2. Configure database connections (MeauxOS DB)
3. Set up API endpoints for dynamic data

## Database

Uses MeauxOS DB for data persistence and synchronization.
